This plugin allow to create a popup with the cart displayed inside.

== Description ==
This plugin display a popup with the cart.
Page option will appear in your WordPress backend menu, and here you can edit quickly your Popup title and menu item label.
You only have to edit this text and save it! Popup will be displayed.

**Important!** Create a **new menu** if not exist!

PREMIUM FEATURES of Popup Cart:

Link to a **new popup** by clicking Add to Cart button.
New **style and effects** for Popup!

== Installation ==

1. Upload the entire 'ns-custom-popup-cart' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. View admin menu, then click on 'NS Popup Cart' and edit label
